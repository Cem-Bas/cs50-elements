CS50 Elements

Version 1.0.0

A powerful Chrome extension for inspecting and analyzing webpage elements. Get detailed information about any element including XPath, CSS selectors, computed styles, and more.

Features

  • Visual element selection with hover highlighting
  • XPath and CSS selector generation
  • Computed styles inspection
  • Element attributes and metadata
  • One-click copy for selectors
  • Clean, developer-friendly interface
